threegen
========

Three.js game engine

[Documentation](https://github.com/codenameyau/threegen/wiki)

##ThreeGen Architecture

###Core Engine
* Timing system
* Keyboard input
* Mouse input
* World model
* Audio system
* Resource loader
* Mathematics
* Utilities

###Rendering Engine
* Three.js renderer
* Animation system
* Camera perspective
* Lighting

###Physics Engine
* Collision detection
* entity movement

##Game
* Documentation
* Scene graph
* Entity (game object)
* Artificial intelligence
* Level editor
