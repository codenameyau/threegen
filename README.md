threegen
========

Three.js game engine.

##ThreeGen Architecture

###Core Engine
* Keyboard input
* Mouse input
* Timing system
* World model
* Artificial intelligence
* Audio system

###Rendering Engine
* Extended on Three.js
* Animation system
* Camera perspective
* Lighting

###Physics Engine
* Collision detection
* Player movement
